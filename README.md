### 구현 기능

**signin.html**

1. 로그인 기능 
2. 회원가입페이지 및 메인페이지 이동

**signup.html**

1. 회원가입기능
2. 회원가입완료시 로그인페이지 이동

**main.html**

1. 검색기능
2. 로그아웃기능
3. 추천기능
4. 각 드라마 상세페이지 이동

**detail.html**

1. 댓글 및 별점 포스팅 및 리스팅
2. 관련 추천 드라마 상세페이지 이동
